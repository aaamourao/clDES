Detailed Description

template<uint8_t NEvents, typename StorageIndex = unsigned int>
class cldes::DESystemCL< NEvents, StorageIndex >

Discrete-Events System on device memory.

Implement a DES on the device memory and their respective operations for GPUs.

Template Parameters
NEventsNumber of events
StorageIndexUnsigned type use for indexing the ajacency matrix
