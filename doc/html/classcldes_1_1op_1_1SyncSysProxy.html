Detailed Description

template<uint8_t NEvents, typename StorageIndex>
class cldes::op::SyncSysProxy< NEvents, StorageIndex >

Parallel composition return type.

Represents a parallel composition. Allows lazy evaluation, but can be converted to a concrete system any time with a DESystem() cast. It is implemented as a binary tree of implicit parallel compositions, where the leafs are concrete systems.

Template Parameters
NEventsNumber of events
StorageIndexUnsigned type use for indexing the ajacency matrix
