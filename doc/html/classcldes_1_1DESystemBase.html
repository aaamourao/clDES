Detailed Description

template<uint8_t NEvents, typename StorageIndex>
class cldes::DESystemBase< NEvents, StorageIndex >

A discrete-events system base abstract class.

DESystemBase implements discrete-events interface and its basic properties.

Template Parameters
NEventsNumber of events
StorageIndexUnsigned type use for indexing the ajacency matrix
